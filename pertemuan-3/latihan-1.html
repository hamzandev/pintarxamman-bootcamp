Ringaksan materi pertemuan kedua

Pada pertemuan kedua, ada beberapa materi yang diberikan. Yaitu mengenai pengenalan HTML dan tag-tag dasar pada HTML. Berikut beberapa ringaksan dari pertemuan kedua :

  1. Pengertian apa itu HTML

    HTML adalah singkatan dari HyperText Markup Language, yang merupakan bahasa markup yang menjadi standar untuk membuat sebuah halaman website

  2. Struktur dasar HTML

    Pada dasarnya html memiliki struktur dasar seperti berikut 

                <!DOCTYPE html>
            <html lang="en">
            <head>
                <title>Title</title>
            </head>
            <body>
                
            </body>
            </html>

  3. Cara penulisan tag HTML

    Pada dasarnya penulisan tag HTML menggunakan tanda "<>" untuk tag pembuka dan juga "</>" untuk tag penutupnya. Perlu diketahui bahwa ada beberapa tag HTML yang tidak memiliki tag penutup seperti hr, br, link, dll

  4. Attribute tag HTML

    Attribute pada tag HTML adalah sesuatu yang dituliskan setelah nama tag di tag pembuka. contohnya : <a href="https://hamzan-wahyudi.vercel.app"></a> kata "href" merupakah attribute tag untuk tag anchor yang mana attribute tersebut memiliki value "https://hamzan-wahyudi.vercel.app"

  5. Tag-tag pada HTML 
                <br/>
            <b></b>
            <i></i>
            <u></u>
            ...dan banyak lagi
  6. Tag gambar pada HTML

    Untuk menampilkan gambar pada HTML, kita bisa menggunakan tag <img src="" alt=""> dan berikan alamat gambar yang ingin ditampilkan pada attribute "src", untuk "alt" berguna jika gambar tidak berhasil atau lama di muat.

  7. Tag List (Daftar)

    Ada 2 jenis tag list yang sering digunakan, yakni ordered list (daftar terurut) dan unordered list (daftar tidak terurut)


    Ordered List : 
                <ol>
                <li>...</li>
                <li>...</li>
                <li>...</li>
            </ol>
    Unordered List : 
                <ul>
                <li>...</li>
                <li>...</li>
                <li>...</li>
            </ul>
  8. Tag table pada HTML

    Pada HTML, kita pasti membutuhkan tabel untuk menampilkan data. Untuk membuat tabel pada HTML, kita bisa menggunakan tag <table></table> lalu diikuti dengan kolom dan baris tabel nya. Untuk membuat kolom tabel, gunakan tag <td></td> dan untuk membuat baris tabel, gunakan tag <tr></tr>. Contoh : 

                <table border="1">
                <tr>
                    <td>Nama</td>
                    <td>Alamat</td>
                    <td>No. HP</td>
                </tr>
                <tr>
                    <td>Hamzan Wahyudi</td>
                    <td>Sumbawa</td>
                    <td>08xxxxxxxxxx</td>
                </tr>
            </table>
